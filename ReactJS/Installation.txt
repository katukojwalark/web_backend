
React JS

Install Node JS from: https://nodejs.org/en/download/

at gitbash from the folder where you want to have your nodejs repository run the following command,
 
    npx create-react-app firstapp  // This is one time process.

 once done with the setup, 

 gitbash at the folder where you want to run your react app (i.e. firstapp) and give the command,

    npm start  //this will run start the browser for you.
